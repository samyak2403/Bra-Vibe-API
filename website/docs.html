Documentation

Welcome to the official Bra Vibe API documentation. This is an Educational Research Project designed to explore data normalization for e-commerce pricing feeds.

Authentication

Currently, the API is **publicly accessible** via GitHub Raw URLs. No API key is required for moderate usage.

Core Endpoints

All requests are standard GET requests returning JSON content.

GET https://raw.githubusercontent.com/samyak2403/Bra-Vibe-API/main/data/bras_deals.json

Automation & Updates

The API is not a static file; it is a live feed managed by our high-performance scraping engine.

  • **Update Frequency**: Synced every **1 hour** automatically.
  • **Platform**: Powered by **GitHub Actions** via `.github/workflows/scrape_deals.yml`.
  • **Filtering**: Only deals with a **minimum 20% discount** are included in the feed.

Full JSON Schema

The response object contains a `metadata` header and a `deals` array.

Field Type Description
product_idStringUnique ID from the source store.
nameStringNormalized product title.
brandStringBrand name or "N/A".
categoryStringPrimary product category (e.g., Push-Up Bra).
price_originalFloatMRP or original listed price.
price_discountedFloatCurrent flash sale price.
discount_percentageIntegerPercentage off (Min 20%).
sizes_availableArray[String]List of available sizes (e.g., ["32B", "34C"]).
colors_availableArray[String]List of available color variants.
image_urlStringHigh-resolution primary product image.
image_urlsArray[String]List of all available product image variants.
product_urlStringDirect deep-link to the store page.
website_sourceStringThe source store (e.g., Amazon, Ajio).
scraped_atStringISO timestamp of the deal discovery.

JavaScript Integration

fetch('https://raw.githubusercontent.com/samyak2403/Bra-Vibe-API/main/data/bras_deals.json')
  .then(res => res.json())
  .then(data => {
    console.log(`Found ${data.metadata.stats.final_count} deals!`);
  });

Java Native Integration

URL url = new URL("https://raw.githubusercontent.com/samyak2403/Bra-Vibe-API/main/data/bras_deals.json");
HttpURLConnection conn = (HttpURLConnection) url.openConnection();
conn.setRequestMethod("GET");

BufferedReader in = new BufferedReader(new InputStreamReader(conn.getInputStream()));
String inputLine;
StringBuilder content = new StringBuilder();
while ((inputLine = in.readLine()) != null) {
    content.append(inputLine);
}
in.close();
System.out.println(content.toString());

Troubleshooting

CORS Issues

GitHub Raw does not altid permit cross-origin requests from some domains. Use a proxy like `cors-anywhere` if you hit restrictions in a browser.

Stale Data

Browser caching may show old results. Append a timestamp as a query parameter (e.g., `?t=12345`) to force a fresh fetch.
